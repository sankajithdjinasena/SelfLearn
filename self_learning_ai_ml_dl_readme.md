# AI / ML / DL Self-Learning Roadmap

A complete self-learning guide for Artificial Intelligence (AI), Machine Learning (ML), and Deep Learning (DL).

---

# 1. Introduction

## What is AI?
Artificial Intelligence (AI) is the field of creating systems that can perform tasks that normally require human intelligence.

Examples:
- Chatbots
- Self-driving cars
- Face recognition
- Recommendation systems
- Voice assistants

## What is Machine Learning?
Machine Learning (ML) is a subset of AI where systems learn from data instead of being explicitly programmed.

Examples:
- Spam email detection
- Stock prediction
- Fraud detection
- Movie recommendations

## What is Deep Learning?
Deep Learning (DL) is a subset of ML that uses neural networks with many layers.

Examples:
- Image classification
- Object detection
- Speech recognition
- Generative AI

# 6. Machine Learning Basics

## Types of Machine Learning

### Supervised Learning
Learning using labeled data.

Examples:
- Classification
- Regression

Algorithms:
- Linear Regression
- Logistic Regression
- Decision Trees
- Random Forest
- SVM
- KNN

### Unsupervised Learning
Learning patterns without labels.

Algorithms:
- K-Means Clustering
- DBSCAN
- Hierarchical Clustering
- PCA

### Reinforcement Learning
Learning through rewards and punishments.

Examples:
- Game AI
- Robotics

# 8. Important ML Concepts

## Training vs Testing Data
- Training data is used to train models.
- Testing data is used to evaluate models.

## Overfitting
Model memorizes training data.

## Underfitting
Model fails to learn patterns.

## Bias vs Variance
Important tradeoff in ML.

## Evaluation Metrics

### Classification
- Accuracy
- Precision
- Recall
- F1-score

### Regression
- MAE
- MSE
- RMSE
- R² score
